# Video URL Cast

Couldn't find what you were looking for?

First, try clicking on the video you want to cast. Some websites defer the loading of content until a user invokes it.

Still nothing? On a page with flash videos, CopyCast can load the page as a mobile web browser and optimistically check to see if the web server at hand decides to serve something more mobile (and chromecast) friendly. This will work in most cases, we'll just pretend to be an iPhone so you can stream your stuff!
